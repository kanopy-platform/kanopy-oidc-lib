# kanopy-oidc-lib

Golang libraries for authenticating with Kanopy.
- A [dex](https://dexidp.io/) identity provider using OIDC (OpenID Connect). 
- Generating personal access tokens and kubeconfigs for Kubernetes clusters.

This public repo is meant to be consumed by user applications who want to use these libraries to authenticate with Kanopy.

> [!NOTE]
> There is a separate private repo that releases the `kanopy-oidc` CLI, that end users can download. Please see kanopy docs for installation and setup.
